🔌 Guide de connexion local (CORS)

Pour faire fonctionner le chatbot en ligne avec votre serveur local, suivez ces instructions :

  1. Quittez Ollama en arrière-plan (clic droit sur la tête de lama près de l'horloge -> Quit Ollama).
  2. Ouvrez PowerShell et exécutez ces commandes : 
    $env:OLLAMA_ORIGINS="*"
ollama serve
  1. Arrêtez le service Ollama existant dans votre terminal : 
    # macOS (si lance via launchd)
launchctl unload ~/Library/LaunchAgents/ollama.plist

# Linux
sudo systemctl stop ollama
  2. Lancez Ollama avec l'autorisation réseau : 
    OLLAMA_ORIGINS="*" ollama serve

⚠️ Action requise sur votre Navigateur

Les navigateurs bloquent les requêtes non sécurisées (HTTP) depuis un site en ligne (HTTPS). Pour débloquer la connexion :

  1. Cliquez sur l'icône de réglages (curseurs ou cadenas) à gauche de l'URL dans la barre d'adresse.
  2. Cliquez sur Paramètres des sites.
  3. Trouvez la ligne Contenu non sécurisé et changez-la en Autoriser.
  4. Revenez ici et actualisez la page (F5).
